## Framework đánh giá dự kiến

### Feature Validator
```
1. User value       — Giải quyết pain point gì? Cho ai?
2. Dev effort       — Mất bao lâu? Phức tạp không?
3. Priority score   — RICE: Reach · Impact · Confidence · Effort
4. Quyết định       — Build now / Later / Skip / Need more info
```

### Product Idea Validator
```
1. Problem          — Vấn đề có thật không? Ai đang chịu đựng?
2. Solution fit     — Giải pháp có khớp vấn đề không?
3. Market           — Thị trường đủ lớn không? Competitor?
4. MVP scope        — Cần build gì tối thiểu để validate?
5. Rủi ro           — Kỹ thuật · Thị trường · Cạnh tranh
6. Quyết định       — Pursue / Pivot / Park
```

### Technical Solution Evaluator
```
1. Feasibility      — Có làm được với stack hiện tại không?
2. Trade-off        — Performance vs Complexity vs Maintainability
3. Scalability      — Chịu tải được không khi scale?
4. Thời gian        — Estimate implement + test
5. Rủi ro kỹ thuật — Dependency · Security · Vendor lock-in
6. Khuyến nghị      — Chọn giải pháp nào và tại sao
```
