# Personal Health Profile

## Description

Tracks personal health baseline, injury history, fitness goals, and recovery status. Helps plan training (especially Pickleball), monitor fatigue, and prevent overtraining. Maintains a living health profile that informs workout and recovery recommendations.

## Triggered When

- User says "update my health profile", "log workout", "I'm injured"
- User says "cập nhật sức khỏe", "ghi lại tập luyện", "tôi bị chấn thương"
- User asks "am I recovered enough to play/train", "tôi đã hồi phục chưa"
- User asks about training plan, rest days, nutrition timing
- User says "lịch tập", "nghỉ ngơi bao lâu", "ăn uống thế nào"
- Weekly or monthly check-in on health goals

## Rules

- Keep profile updated after every significant health event
- Be conservative with injury — one extra rest day is always better
- If pain > 5/10 during activity: STOP immediately
- Use the recovery protocol BEFORE returning to full training
- Goals should be S.M.A.R.T. — Specific, Measurable, Achievable, Relevant, Time-bound
